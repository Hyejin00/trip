# trip
2-2학기 가초 웹프로그래밍의 기말 프로젝트인 여행 상품 판매 사이트.

https://sheltered-tor-58539.herokuapp.com/
## 사용 기술
html(pug), bootstrap, javascript, AWS S3, mongoDB, heroku
## 기능
* 사용자
  * 여행 상품 읽기, 검색, 예약
  * 상품 게시글에 댓글 생성, 삭제
  * 예약 상품 읽기, 삭제
  * 마이페이지(나의 정보) 수정, 회원 탈퇴
* 여행 가이드
  * 사용자 기능
  * 나의 상품 예약 읽기, 삭제
  * 나의 상품 읽기, 삭제, 생성
* 관리자
  * 사용자 기능
  * 이용자 읽기, 삭제, 권한 설정
  * 예약 목록, 상품 목록, 리뷰 읽기 및 삭제
